## ğŸ’¾ Storage Locations

### Local Database (SwiftData)
- **Location:** App container
- **Content:** All workouts, sets, exercises
- **Backup:** Automatic via iCloud
- **Size:** Typically < 50MB even with years of data

### User Defaults (AppStorage)
- **Location:** App preferences
- **Content:** Settings, flags, favorites
- **Backup:** Automatic via iCloud
- **Size:** < 1MB

### Temporary Files
- **Location:** Temp directory
- **Content:** Export files before sharing
- **Cleanup:** Automatic by iOS

### iCloud Container
- **Location:** CloudKit private database
- **Content:** Synced workout data
- **Encryption:** Apple-managed
- **Size:** Counts against user's iCloud quota
